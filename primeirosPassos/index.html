Ferramentas Utilizadas

Na abordagem inicial da formação, utilizamos principalmente o Visual Studio!

O Visual Studio Code (VS Code) é uma das IDEs mais populares e eficientes para o desenvolvimento de softwares. O que diferencia o VS Code de outras IDEs é a sua simplicidade e flexibilidade. Ele é um editor de código aberto e multiplataforma, compatível com diversas linguagens de programação como Python, JavaScript, C++, entre outras. Além disso, possui inúmeras extensões gratuitas que podem ser instaladas para tornar o ambiente de trabalho ainda mais poderoso e personalizado. Com o VS Code, os desenvolvedores têm a possibilidade de usar recursos avançados, como debugging, git integrado, realce de sintaxe e muito mais, de forma intuitiva e eficiente. Em suma, o VS Code é uma ferramenta essencial para qualquer desenvolvedor em busca de produtividade e agilidade no seu workflow.

Além disso, utilizamos um navegador para a interpretação dos arquivos HTML. Por preferência e gosto próprio, eu utilizei o Chrome, que é bastante eficiente e intuitivo.

Inspetor de Elementos

Para melhor compreensão dos elementos HTML organizados nas páginas WEB, utilizamos nas aulas o Inspetor de Elementos presente na maioria dos navegadores.

O Inspetor de Elementos do Chrome é uma ferramenta de desenvolvedor integrada ao navegador Google Chrome que permite aos desenvolvedores e designers inspecionar e editar o código HTML, CSS e JavaScript de páginas da web. Com ele, é possível visualizar a estrutura do documento HTML, bem como todas as propriedades de estilo aplicadas aos elementos, além de possibilitar a edição dinâmica desses códigos. Além disso, o Inspetor de Elementos do Chrome também oferece recursos para depuração de erros, como a capacidade de verificar elementos da página que não estão funcionando corretamente ou que estão gerando erros. Essa ferramenta é extremamente útil para desenvolvedores web que desejam criar sites responsivos e compatíveis com diferentes dispositivos e navegadores.

Estrutura do HTML

O HTML é a linguagem de marcação usada para construir a estrutura de páginas da Web. Seu uso permite criar uma estrutura hierárquica e semântica do conteúdo de uma página, tornando mais fácil o entendimento do seu significado tanto para os desenvolvedores quanto para os usuários. O HTML é composto por elementos que são representados por tags. Essas tags são utilizadas para definir a estrutura básica da página, como cabeçalho, parágrafos, imagens, listas, formulários, entre outros. É fundamental entender a estrutura do HTML para criar uma página com uma boa organização de conteúdo, possibilitando uma melhor acessibilidade e usabilidade para seus usuários.

TAGS

As TAGS do HTML, ou HyperText Markup Language, são elementos que permitem estruturar e formatar o conteúdo de uma página web. Cada TAG é composta por um código delimitado por sinais de menor e maior, e pode conter atributos que definem suas características.

Algumas das TAGS mais comuns e utilizadas em páginas web são:

  1. <html>: define o início do documento HTML e contém todas as outras TAGS;
  2. <head>: define informações sobre o documento, como título, links para estilos CSS e scripts JavaScript;
  3. <body>: define o corpo do documento, onde fica o conteúdo visível da página, como textos, imagens e vídeos;
  4. <a>: define um link, que pode direcionar o usuário para outra página web, arquivo, ou posição na mesma página.
  5. <ul>: é utilizada para criar uma lista não ordenada e seus itens são definidos pela tag "li".

Listas (Ordenadas e Não Ordenadas)

As tags de lista em HTML são fundamentais para organizar e estruturar conteúdo em uma página da web. Existem dois tipos de listas: as listas ordenadas e as listas não ordenadas.

As listas ordenadas são criadas com a tag "<ol>", que define uma lista numerada. Cada item da lista é definido pela tag "<li>". O número correspondente a cada item é definido automaticamente pelo navegador, iniciando em 1 por padrão, mas pode ser alterado utilizando o atributo "start" na tag <ol>.

Já as listas não ordenadas são criadas com a tag "<ul>", que define uma lista sem ordem específica. Cada item da lista também é definido pela tag "<li>". Nesse caso, o navegador utiliza um marcador (como um ponto ou um círculo) para identificar cada item da lista.

Ambos os tipos de listas podem ser aninhados, permitindo a criação de listas dentro de outras listas. As tags de lista são muito úteis para exibir informações em formato de tópicos, como listas de itens, passos de instruções, características de produtos, entre outros.

Links

Os links em páginas HTML são utilizados para conectar diferentes páginas da web entre si, permitindo que o usuário navegue de uma página para outra com facilidade. Os links são criados utilizando a tag "<a>", que possui um atributo "href" indicando o endereço da página de destino.

Os links podem ser exibidos como texto ou como imagens. Quando são exibidos como texto, o usuário pode clicar no texto para acessar o link. Já quando são exibidos como imagens, o usuário pode clicar na imagem para acessar o link. É importante lembrar que os links devem ser descritivos e indicar claramente o conteúdo da página de destino. Além disso, é recomendável utilizar títulos descritivos nos links para facilitar a navegação do usuário e melhorar a acessibilidade da página.

Os links também podem ser utilizados para criar ancoragens dentro de uma mesma página, permitindo que o usuário navegue para diferentes seções da página. Para criar uma âncora, é necessário definir um identificador único para a seção desejada, utilizando a tag "<a>" com o atributo "id". Em seguida, basta adicionar um link com o atributo "href" apontando para esse identificador precedido pelo caractere "#".
